# camera_calibration
このレポジトリではカメラのキャリブレーションを行うことができます

## 動作確認済環境
os : Ubuntu 22.04 lts  
OpenCV : 4.5.4

## インストール方法

リポジトリをクローン
```
git clone https://github.com/TakumiIshiguro/camera_calibration.git
```
ディレクトリを移動
```
cd camera_calibration
```
## 実行方法

１．キャリブレーションを行うための画像を10枚用意  
２．```cal.py```内のファイルパスを変更  
３．キャリブレーションを実行
```python3 cal.py```
